# Test Cases Contract

## 测试用例规范

所有示例程序必须通过以下测试用例：

### 1. 正常加法
- 输入: A=10, B=20
- 预期输出: Result=30

### 2. 负数加法
- 输入: A=-5, B=3
- 预期输出: Result=-2

### 3. 正向溢出
- 输入: A=32767, B=1
- 预期输出: Result=-32768（回绕）

### 4. 负向溢出
- 输入: A=-32768, B=-1
- 预期输出: Result=32767（回绕）

### 5. 零值
- 输入: A=0, B=0
- 预期输出: Result=0
